## 🚀 Projects

### 🫁 MediScan AI — Chest X-Ray Disease Detection
> **Production-grade medical AI with Explainable AI + RAG Clinical Assistant**

Chest X-ray pneumonia detection system using DenseNet-121 fine-tuned on 5,863 real X-rays. Features Grad-CAM heatmaps for explainability and a RAG-powered clinical assistant grounded in WHO guidelines.

**Results:** Test AUC-ROC: **0.96+** · Test Accuracy: **90.2%**

**Technologies:** `Python` `PyTorch` `DenseNet-121` `Grad-CAM` `LangChain` `FAISS` `LLaMA-3.3-70B` `Groq API` `Streamlit` `HuggingFace`

[![Live Demo](https://img.shields.io/badge/🔴_Live_Demo-FF4B4B?style=for-the-badge)](https://mediscan-ai-jzmgxsbzhstdazbwtsmch6.streamlit.app/)

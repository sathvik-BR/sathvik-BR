# 🚀 Projects

### 🧠 AI Knowledge Workspace (RAG + LLM)

AI-powered workspace that allows users to interact with PDF documents using Natural Language Processing and Retrieval-Augmented Generation (RAG).

Users can upload documents, ask questions, perform semantic search, generate summaries, and create AI-powered study flashcards.

**Technologies:** Python, Streamlit, LangChain, FAISS, HuggingFace Embeddings, Groq API, NLP

### 📄 AI Resume Analyzer

Developed an AI-powered web application that analyzes resumes and compares them with job descriptions using Natural Language Processing.

The system calculates a **resume match score**, identifies **missing skills**, and visualizes **skill coverage** to help job seekers improve their resumes.

**Technologies:** Python, NLP, Scikit-learn, TF-IDF, Cosine Similarity, Streamlit
